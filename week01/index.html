Master Mobile UX Notes

  • After 2011, mobile devices really took off and designing a disply for this changed drastically because of it.
  • Eric Shcmidt said in 2012, "The simple guidline is whatever you are doing - do mobile first."
  • Responsive Retrofitting is useful if you already have a desktop web version of your site and need to quickly get it to mobile, but ultimatley you will need to rewrite it for mobile first to better include touch interaction and address screen real estate issues.
  • Doing mobile first helps build for a more friendly user experience and allows for easier growth of the website. It also constrains bandwidth to make your site work faster and more efficiently.
